# Husan Ali Premium Linkhub - Ishga Tushirish va Boshqarish Qo'llanmasi

Ushbu loyiha (sayt) oddiy va zamonaviy HTML, CSS hamda JavaScript asosida qurilgan. Shu sababli, uni ishga tushirish uchun hech qanday murakkab dasturlar yoki serverlar shart emas.

Quyida saytingizni qanday ochish, tahrirlash va kelajakda Internetga chiqarish haqida qadam-ba-qadam ko'rsatmalar yozilgan.

---

## 🚀 1-usul: Eng oson va tezkor yo'l (Odatiy ko'rish)
Sayt tayyor va hamma narsa bitta qutining ichida turibdi. Saytingizni kompyuterda ko'rish uchun quyidagilarni bajaring:

1. **Papkaga kiring:** Kompyuteringizdan `d:\Documents\husan-ali-portfolio` papkasini oching.
2. **Faylni toping:** Papka ichidagi `index.html` degan faylni toping.
3. **Saytni oching:** Shu `index.html` faylining ustiga ikki marta bosing. (Yoki o'ng tugmani bosib, "Open with" -> Chrome / Edge brauzerini tanlang).
4. **Natija:** Brauzeringizda sizning shaxsiy Linkhub saytingiz ochiladi!

---

## 🌐 2-usul: Lokal Server orqali ochish (Professional yo'l)
Agar kodingizni qachondir o'zgartirsangiz va eng so'nggi natijalarni tutilishsiz ko'rmoqchi bo'lsangiz, men yozib bergan kichik lokal server orqali ishga tushirishingiz mumkin.

1. **Papkaga kiring:** `d:\Documents\husan-ali-portfolio` jildiga kiring.
2. **Server skriptini toping:** O'sha yerdagi `server.ps1` faylini toping.
3. **Ishga tushiring:** `server.ps1` fayli ustida sichqonchaning o'ng tugmasini bosing va menyudan **"Run with PowerShell"** (PowerShell dasturida ishga tushirish) ni bosing. Quyida qora/ko'k oyna ochiladi va server ishlashni boshlaydi.
4. **Brauzerga yozing:** Endi istalgan brauzerni (Chrome, Safari va h.k) ochib tegishli manzil bo'limiga `http://localhost:8080/` deb yozing va kiring.

*Eslatma: Bu xuddi saytingizni haqiqiy internetdagi kabi simulyatsiya qilib beradi.*

---

## 🛠️ Saytga qanday qilib o'zgartirish kiritiladi?
Agar kelajakda ismingizni, maqomingizni yoki biror linkni o'zgartirmoqchi bo'lsangiz:

1. `index.html` fayli ustiga o'ng tugmani bosing.
2. Menyudan **"Open with"** -> **"Notepad"** (Yoki kompyuterda Visual Studio Code bo'lsa, o'shani) tanlang.
3. Kerakli so'zni topib, o'rniga yangisini yozing.
4. Klaviaturada **Ctrl + S** bosib saqlang.
5. Brauzerda saytga o'tib, u yerni ham birmarta yangilang (F5 tugmasi orqali). Muvaffaqiyatli!

---

## 🌍 Internetga tekinga joylashtirish (Host qilish) haqida
Ushbu "Linkhub" faqat kompyuteringizda turadigan narsa emas. Siz bu kodni global internetga joylab ommaviy qilishingiz mumkin:

Buning qadamlari (netlify.com orqali misol):
1. **Netlify.com saytiga kiring** va bepul akkaunt oching.
2. Bosh sahifasida ko'rinib turgan **"Drag and drop your site folder here"** degan joyni toping.
3. O'zingizdagi kompyuteringizda bo'lgan `husan-ali-portfolio` papkasini sichqoncha bilan tutib, o'sha saytning ekraniga tortib tushiring.
4. Saytingiz shu ondayoq internetga yuklanadi! Sizga ulashishingiz uchun havola (Masalan: *husan-ali-linkhub.netlify.app*) taqdim qilinadi.
